# Hands-On Experience Implementing Quantum Computing (IBM QPU + VS Code)

A **3-hour, absolute-beginner** workshop where you learn quantum circuits by doing:  
**build → simulate → run on a real IBM Quantum QPU → interpret results**.

This repo is organized for **VS Code** and **notebooks** (cells with GOAL + code + exercises).

---

## What you will learn
- Install and use **VS Code** for Python + notebooks :contentReference[oaicite:0]{index=0}
- Set up **Qiskit** + **qiskit-ibm-runtime** using a clean virtual environment :contentReference[oaicite:1]{index=1}
- Understand **qubits** using the **Bloch sphere**
- Use key gates: **X, H, Z, CX**
- Run a **Bell state on a real IBM QPU** using **SamplerV2 (V2 primitives)** :contentReference[oaicite:2]{index=2}
- Mini-protocols:
  - **Quantum One-Time Pad (QOTP):** encrypt/decrypt a qubit
  - **Superdense coding:** send **2 classical bits** using **1 qubit + entanglement**
